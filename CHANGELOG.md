# EcoPaste 重构版更新说明

这个版本对 EcoPaste 的底层架构进行了全面重构。核心目标是让应用更轻、更快、更稳定，同时减少历史实现带来的复杂度。

## 🏗️ 架构重构

- 核心逻辑全面迁移至 Rust，前端更专注于界面展示与交互
- 重构数据存储与运行架构，提升稳定性和可维护性
- 精简旧实现中的复杂路径，降低资源占用和后续维护成本

## ✨ 新特性

- Windows 平台窗口显示时不再抢占当前焦点
- 新增来源应用识别与按应用过滤，可查看历史记录来源，并灵活控制哪些应用的内容不进入剪贴板历史
- 新增按内容类型采集开关，可分别控制纯文本、HTML、RTF、图片、文件和文件夹是否进入历史
- 新增采集顺序设置，可调整文件、图片、HTML、RTF、纯文本在多格式剪贴板内容中的保存优先级
- 新增敏感内容保护，可识别私钥、常见服务 Token、AWS Key、JWT 等高置信凭据并跳过保存
- 新增完整内容预览窗口，支持悬停预览或按住空格键预览文本、图片和文件记录
- 支持将文本、富文本、图片和文件记录直接拖出到外部应用
- 新增更细粒度的复用行为：
  - 左键可配置为禁用、单击/双击粘贴、单击/双击复制
  - 中键可配置为粘贴、粘贴为纯文本、复制、复制为纯文本
  - 文件记录可默认粘贴为文件路径，文本记录可默认复制或粘贴为纯文本
  - 可选择从历史复制后自动隐藏剪贴板窗口
- 新增历史列表排序方式，可按创建时间、复用时间或使用频率浏览记录
- 新增列表展示控制，可调整文本最大显示行数、图片缩略图高度和文件显示数量
- 新增剪贴板条目悬停快捷动作，可在列表项上快速执行粘贴、复制、收藏、备注、删除、打开链接、定位文件等操作，并在偏好设置中调整显示顺序与启用项
- 新增收藏项删除保护设置，可默认禁止删除收藏记录；开启后可限制仅在“收藏”分组显示删除入口，并可单独控制普通记录与收藏记录的删除前确认
- 新增偏好设置搜索，可快速定位具体设置项
- 新增本地存储管理，可查看数据占用、打开数据/日志目录，并清理未被历史记录引用的资源缓存
- 新增自定义数据目录，可迁移历史数据库、设置文件和资源缓存，迁移后清理旧数据，并支持一键还原默认目录
- 增强备份导入导出能力：
  - 使用 `.ecopastebak` 专用备份包，默认密码加密，也可主动选择明文 ZIP 备份
  - 支持单击选择文件、双击打开备份文件，或将备份拖入偏好设置窗口导入
  - 支持合并导入与覆盖导入：合并导入保留当前数据并补齐缺失内容，覆盖导入可热替换历史、资源与设置，无需重启应用
  - 自动识别加密容器与明文 ZIP 备份；导入加密备份时需要输入导出时设置的密码
- 新增重置偏好设置能力，可恢复默认设置并保留历史记录

## 🚀 性能优化

- 重构剪贴板监听机制，复制内容响应更快、更稳定
- 优化整体运行效率，降低资源占用
- 提升应用启动速度和日常使用流畅度
- 优化长文本列表展示，避免大段内容导致应用卡顿
- 搜索迁移到 SQLite FTS5，并覆盖正文与备注内容
- 图片缩略图、应用图标和文件图标由 Rust 侧缓存和复用，减少前端重复解析与渲染成本
- 历史列表分页时一次返回列表、总数和是否还有更多数据，减少额外请求

## 🐛 问题修复

- 修复 Windows 平台以管理员权限运行时开机自启动失效的问题

## ⚠️ 升级须知

### 不兼容旧版本

由于底层架构已经重构，本版本不兼容旧版运行环境及历史数据。

### 支持平台调整

本重构版仅支持 macOS 与 Windows，不再支持 Linux。

### 请务必全新安装

安装前请提前备份重要配置和数据，卸载旧版本后再安装本版本。

### 测试版本

当前版本仍处于测试阶段，可能存在未知问题，建议仅用于尝鲜体验或参与测试反馈。
